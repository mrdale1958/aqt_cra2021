

The electron-js-kiosk folder: This starts up the AQT app in Electron Kiosk Mode
	1. Edit the main.js file
	2. Scroll to Line 13 "mainWindow.loadURL('http://localhost:8089/')" 
	3. If the URL to your app has changed - Substitute the 'http://localhost:8089/' with your URL.If not leave the file as is. 
	4. Save.


AQT Table App - Electron.js: This is used to automate the launching AQT app. Double clicking it will lauch the AQT App.

THE AQT TABLE - A shortcut to the automation script that can be placed on the desktop.
