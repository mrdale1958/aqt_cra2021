1. The aqtserverstart.bat file: Automates the process of launching the AQT Server. 
	a. Edit the .bat file to update the path the AQT folder
	b. Line 3, "cd C:\XAMP\htdocs\AQTTable" 
	c. If path to folder has changed - Substitute 'C:\XAMP\htdocs\AQTTable' with the updated path to the AQT Folder. 
2. THE AQT SERVER : A shortcut to the automation script taht can be placed on the desktop.
