STEP 1:
INSTALL Electron js: 
# Clone the repository
$ git clone https://github.com/electron/electron-quick-start
# Go into the repository
$ cd electron-quick-start
# Install dependencies
$ npm install

STEP 2:
LAUNCH AQT SERVER : Contains shortcut and start up scripts for the AQT Server.

STEP 3:
LAUNCH AQT APP: Contains the shortcut and scripts that launch the AQT app in Electron js kiosk mode.
